# agent-skills

Repository for general AI [Agent Skills](https://cursor.com/docs/context/skills), [Cursor Rules](https://cursor.com/docs/rules) and
[Subagents](https://cursor.com/docs/subagents) that I use in my hobby projects.

- `/agents`: Subagents for multi-agent TDD workflow
- `/rules`: Project specific architecture, conventions etc. Change according to your project!
- `/skills`: Agent skills for specialized tasks
